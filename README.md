# Laravel Docker Example

## Requirements
 * PHP 7.3
   * Laravel 5.8
 * Docker server or desktop

## Essential Files
 * Dockerfile
 * docker-compose.yml
 * .dockerignore
 * docker(folder)
   * vhost.conf
   * web.dockerfile
   
## How to use

```bash
docker-compose build
docker-compose up
```
